3. [依赖注入](./03-依赖注入.md) - DI 容器和依赖注入机制

### 进阶篇

16. [项目结构](./16-项目结构.md) - 项目组织规范

## 🚀 学习路径

1. **初学者**: 从快速开始 → 核心概念 → 依赖注入
2. **进阶者**: 学习中间件、异常处理、管道、守卫、拦截器
3. **实战者**: 数据库集成 → RESTful API → 认证授权
4. **高级**: 微服务、WebSocket、项目结构优化
